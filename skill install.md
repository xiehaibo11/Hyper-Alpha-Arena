1. 均值回归定律（Mean Reversion）

• 核心逻辑：价格不会永远偏离中枢，偏离幅度可通过数据丈量，将市场的情绪冲高或恐慌下跌转化为统计距离。

• 交易机会：当价格冲出常态区间时，模型等待偏离收敛，利用市场的过度反应寻找交易机会。

2. 趋势跟随策略（Trend Following）

• 核心逻辑：不猜测顶底，仅通过数据确认市场方向。观察价格、成交量和波动结构，趋势扩散时顺势加仓，动能衰减时快速退出。

• 交易本质：交易的是市场正在形成的方向，而非主观观点。

3. 动量策略（Momentum）

• 核心逻辑：捕捉短期“强者恒强”的市场特征，动量扩张时跟随冲击，速度下降时收缩仓位。

• 交易本质：交易市场共识形成后的推进力。

4. 凯利公式（Kelly Formula）

• 核心逻辑：通过胜率、赔率和失败概率计算下注比例，优势越大仓位越高，风险越高仓位越克制。

• 作用：用资金增长率约束每一次下注，避免情绪化仓位决策。

5. 马科维茨组合理论（Markowitz Portfolio Theory）

• 核心逻辑：不挑选最强资产，而是通过计算资产相关性、协方差衡量风险抵消，再用权重优化寻找有效前沿。

• 目标：组合出更稳定的收益结构，通过资产相关性分散风险。
1. 正态分布（Normal Distribution）

• 作用：作为资产收益率模型的基础假设，以其钟形曲线锚定市场波动的中轴线，将市场无序的涨跌收敛为概率的致密云团。

• 价值：过滤掉绝大多数无意义的随机噪音，帮助量化交易者理解资产收益的概率分布，为风险衡量（如VaR、置信区间）提供基础。

2. 泊松分布（Poisson Distribution）

• 作用：用于收益率中的跳跃事件假设，以脉冲刻度丈量“黑天鹅”等极端事件的降临频率。

• 价值：将离散的极端冲击编码为可量化的概率矩阵，帮助提前预判市场的跳空与反转，应对尾部风险。

3. 学生T分布（Student's t-distribution）

• 作用：针对资产收益率的厚尾假设，其“肥尾”特性可撕开正态分布的虚假安全区。

• 价值：将被低估的极端风险暴露在概率的聚光灯下，守护资产免受尾部暴击，更真实地刻画市场极端波动的概率。

4. 协方差（Covariance）

• 作用：用于资产收益率模型的联动假设，以矩阵脉络编织资产的联动网络。

• 价值：捕捉板块间的共振与背离信号，用相关性的“剪刀”剪断系统性风险的传导链条，助力构建分散化的投资组合。

5. 贝叶斯定理（Bayes' theorem）

• 作用：作为资产收益率模型的动态信念假设，通过迭代引擎实时刷新市场的先验概率。

• 价值：将每一条新信息融化为认知升级的燃料，在不确定性迷雾中锁定最可能的演化路径，实现对资产收益的动态、区间化预测。
1. 止损线的本质

止损线不是单一价格点，而是价格附近的一串触发卖单。量化交易不预测涨跌，而是丈量市场冲击。

2. 泊松到达模型（Poisson Arrival Model）

• 用泊松分布计算“黑天鹅”类极端事件的到达频率，触发数量由到达强度\lambda(t)的上升速度决定。当市场情绪沸腾时，\lambda(t)会陡升，意味着止损触发的概率快速增加。

3. 流动性密度积分

计算止损线附近可吸收的订单厚度Q。订单越拥挤（Q越薄），止损线越清晰，越容易被打穿。

4. 触发链条

当市场冲击量大于止损线附近的订单吸收量（impact > Q），触发卖单会递归释放，导致价格加速穿破止损线。

5. 核心逻辑

量化交易不是预测方向，而是通过计算触发链条的脆弱点（即\lambda(t)陡升、Q变薄、冲击量超吸收量的区域），精准打穿市场最脆弱的止损防线。

1. 几何布朗运动：价格的随机扩散引擎

• 它将价格趋势拆解为确定性漂移（代表趋势方向）和随机布朗噪声（代表不确定性冲击），用公式  dS = \mu S dt + \sigma S dW  描述价格的随机扩散。

• 价格并非沿单一路径运动，而是在无数可能路径中扩散。其中，漂移决定价格中枢，波动率（\sigma）决定未来价格的扩散宽度，随机冲击持续重塑价格的未来分布。

2. 隐马尔可夫模型（Hidden Markov Model）：从收益推断市场状态

• 承认市场“真实状态不可见”，只能通过收益、成交量、波动率等可观测数据推断隐藏状态（如平静、趋势、冲击）。

• 每新增一根数据，模型会重新刷新状态概率，并非预测涨跌，而是判断市场更接近哪种“状态模式（regime）”，以此为仓位、风控和止损提供依据。

3. 马科维茨均值-方差模型：组合视角的收益与风险

• 不迷信单资产高收益，而是将多资产放入同一坐标系，用期望收益衡量“进攻性”，用方差/协方差衡量“风险代价”。

• 通过优化资产权重，找到“单位风险下收益最高”的有效组合，将投资的“贪婪、恐惧和分散化”转化为可计算的优化问题。

4. 协整性与统计套利：交易价差偏离，而非方向

• 寻找长期关系稳定的资产对，即使短期价格“分岔”（一快一慢），但价差（spread）会长期回归均值。

• 当价差的Z-score突破阈值时，对“高估资产做空、低估资产做多”，通过对冲方向风险，赚取价差回归的收益，实现市场中性套利。

5. GARCH波动率模型：捕捉波动率的“记忆性”

• 波动率不是常数，市场冲击会留下“记忆”。一次剧烈冲击会被纳入条件方差，与上一期波动率共同形成新的风险状态（公式  \sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2 ）。

• 这导致“波动率聚集”：平静行情会延续，恐慌行情也会延续。模型通过识别波动率的聚集模式，判断市场是否进入“危险状态”，进而调整仓位、止损和杠杆。
